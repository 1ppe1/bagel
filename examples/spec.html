MVP Review Fixture

Docksync turns review comments into agent context.

Publish a local HTML artifact, collect comments in the browser, and pull them back into a coding agent workflow.

Demo operator note: click the headline, workflow list, or Anchor Rebase section to create a review comment.

Workflow

  1. Push a single HTML file.
  2. Review the rendered artifact in a sandboxed iframe.
  3. Pull open comments into local context.

Anchor Rebase

Existing comments should reattach to matching DOM elements after a safe edit, or become orphaned when the target disappears.

For the orphan demo, remove this entire section and push the file again.
